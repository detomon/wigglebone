# Changelog

## v3.0.0 - 2025-11-11

### Changed

- Rewrite modifier as `SkeletonModifier3D`
- Use spring simulation instead of verlet integration

### Deprecated

- `WiggleBone` and `WiggleProperties` should not be used anymore and will be removed in a later version

## v3.1.0 - 2025-12-12

### Changed

- Update class documentation

### Added

- Allow to set a list of bones for `DMWBWigglePositionModifier3D` and `DMWBWiggleRotationModifier3D` instead of only a single one
